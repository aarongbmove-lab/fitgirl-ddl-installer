# fitgirl-ddl-installer
A super simple FitGirl downloader/installer. Python powered

A Python app that makes fitgirl games really easy to download and install. The app allows you to search for, view info on, and automatically download all individual .rar files, extract them, and then start the installation process.

1. [Download Latest Release](https://github.com/aarongbmove-lab/fitgirl-ddl-installer/releases/download/v1.0.0/FitGirlDDLHelper.zip)
2. Extract the folder
3. Install dependencies first by launching the dependency installer shortcut (launches a bash script to check for python and dependencies and install if needed)
4. Start the app with the shortcut (launches a bash script to check for python and dependencies before running the python script.)

All rights to fitgirl and all of their files, website elements and information are theirs and theirs alone. This is only a simpler way to install and download their games.
